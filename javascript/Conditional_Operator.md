条件 (三項) 演算子は JavaScript では唯一の、3 つのオペランドをとる演算子です。
この演算子は、if 文のショートカットとしてよく用いられます。

## 構文
```
condition ? expr1 : expr2
```

## 引数
### condition
true か false かを評価する式です。
### expr1, expr2
各々の値の場合に実行する式です。
trueならexpr1、falseならexpr2
